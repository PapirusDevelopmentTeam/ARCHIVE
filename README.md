# Install
**Archlinux (AUR)**:
```
yaourt -S papirus-bomi-skin
```
**Kubuntu 14.04/16.04**:
```
sudo add-apt-repository ppa:varlesh-l/papirus-pack
sudo apt-get update
sudo apt-get install bomi-skin-papirus
```
